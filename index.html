Projects

Outside of the classroom, I like to design software for myself and others to use. Below are some of the projects I've worked on with all code available on Github.

GII-481

While alternate text is not required for commercial or informational web pages, visually-impaired surfers miss out on both contextual information(product pages) as well as entertainment(memes) due to minimal use of alt-attributes on the web. Myself and three of my classmates teamed up to design a Google Chrome Extension to address these concerns. Our extension makes use of Google's Cloud Vision API and Microsoft's Computer Vision API to add both image descriptions as well as OCR as alt-attributes of images on the web. Source code can be found on Github.

Icon App Server Consultant

Icon Promotional is a southern-California based start-up that works off of Apple's new directive to open up their native iMessage application to third-party developers by delivering iMessage stickers based on geo-location. I signed onto the project as a part-time consultant and used Python's Django framework to implement the server that determines which stickers are available to the user based on their location. Source code not publically available.
